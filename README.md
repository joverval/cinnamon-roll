# cinnamon roll

A personal [Strudel](https://strudel.cc) player. Live coding music in the browser: TidalCycles port to JavaScript.

**joverval.cl/cinnamon-roll**

## Usage

Open the page, write patterns, hit **Play** (or Ctrl+Enter).

```js
s("bd sd").jux(rev)
```

Click a preload from the sidebar to load a saved pattern.

## Adding preloads

1. Add a `.js` file to `preloads/`
2. Add an entry to `preloads/manifest.json`
3. Push

## Samples

Curated subset of [Dirt-Samples](https://github.com/tidalcycles/dirt-samples) stored in `samples/`. All processing happens in your browser, no server needed.

## Deployment

GitHub Pages on `joverval.cl/cinnamon-roll` (project pages inherit the domain from joverval/joverval).
